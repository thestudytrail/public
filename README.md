# The Study Trail Public Repository

This repository holds any public information from The Study Trail.  If you have a feature request or a bug to file, feel free to do so here.  We ask you follow any applicable guidelines below.

# Feature Requests

To file a Feature Request please do the following:
1. Check if another feature reqeust already covers your idea.  If so, add your thoughts there instead.
2. On the top, click **Choose a different template** then **Feature request**
3. Be as specific as possible.  Examples of how you would use the feature are extremely helpful.
4. Include what you do now to accomplish the same thing.  This also lets us understand the ask better.
5. Check back for any follow up questions we have.

# Bug Tickets

To file a Bug please do the following:
1. On the top, click **Choose a different template** then **Feature request**
2. Include **Specific** instructions for reproduction.  "I opened an assignment" is usally not specific.  "I was on the todolist page, clicked on an assignment, and was instead sent to login" would be specific.
3. Include what you expected to happen.
4. Include a link to the faulty page, or the last succesful page if the bug is on a click.
